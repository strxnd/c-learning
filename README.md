# C Programming Curriculum (K&R 2nd Edition)

## Duration: 8 Weeks
**Book**: *The C Programming Language*, 2nd Edition, Brian Kernighan & Dennis Ritchie  
**Goal**: Complete and master ANSI C through every chapter, with projects and exercises

---

## Week 0 – Setup & Warm-up (Optional)

### Objectives:
- Install `gcc`, `gdb`, editor
- Learn to compile and debug
- Setup GitHub repo for all weekly work

### Deliverables:
- Hello World program
- Setup `.c`/`.h` file structure

---

## Week 1 – Control Flow

### Reading:
- Chapter 3: **Control Flow** — pp. **55–76**

### Topics:
- `if`, `else`, `while`, `for`, `do-while`
- `switch`, `break`, `continue`, `goto`

### Exercises:
- 3-1 to 3-6 — pp. **76–77**

### Mini Projects:
- CLI menu-based program
- Prime checker
- Temperature table with custom step

### Milestone:
- Comfort with conditional logic and structured flow

---

## Week 2 – Functions and Program Structure

### Reading:
- Chapter 4: **Functions and Program Structure** — pp. **79–99**

### Topics:
- Function declarations, scope
- `static`, `extern`, `register`
- Header files, recursion

### Exercises:
- 4-1 to 4-13 — pp. **99–101**

### Mini Projects:
- Recursive factorial, `power()`, and `reverse()`
- Create a header file and reuse across multiple `.c` files

### Milestone:
- Can structure C programs modularly and use scope properly

---

## Week 3 – Pointers and Arrays (Part 1)

### Reading:
- Chapter 5 (Part 1): **Pointers and Arrays** — pp. **101–116**

### Topics:
- Pointers & arrays
- Pointer arithmetic
- Strings as `char*`

### Exercises:
- 5-1 to 5-8 — pp. **116–117**

### Mini Projects:
- Custom implementations: `strlen`, `strcpy`, `strcmp`
- String tokenizer

### Milestone:
- Grasp on pointer fundamentals and string manipulation

---

## Week 4 – Pointers and Arrays (Part 2)

### Reading:
- Chapter 5 (Part 2): **Pointers to Functions, Arrays of Pointers** — pp. **116–135**

### Topics:
- Arrays of pointers
- Function pointers
- Sorting with `qsort`

### Exercises:
- 5-9 to 5-17 — pp. **135–136**

### Mini Projects:
- Stack/Queue using pointers
- Function pointer-based dispatcher

### Milestone:
- Mastered deep pointer logic and memory-safe usage

---

## Week 5 – Structures

### Reading:
- Chapter 6: **Structures** — pp. **137–154**

### Topics:
- `struct`, nested structs
- `typedef`, `enum`
- Linked lists, arrays of structs

### Exercises:
- 6-1 to 6-5 — pp. **154**

### Mini Projects:
- Contact manager
- Student record system
- Linked list for strings

### Milestone:
- Build and use complex data structures

---

## Week 6 – Input and Output

### Reading:
- Chapter 7: **Input and Output** — pp. **156–174**

### Topics:
- `getchar`, `putchar`
- `fgets`, `fputs`, `fopen`, `fclose`
- `scanf`, `printf`

### Exercises:
- 7-1 to 7-8 — pp. **174**

### Mini Projects:
- File copier (`cp`)
- CSV reader
- Word and line counter

### Milestone:
- Fluent in handling file and user input/output

---

## Week 7 – UNIX System Interface (Advanced)

### Reading:
- Chapter 8: **The UNIX System Interface** — pp. **179–202**

### Topics:
- File descriptors: `open`, `read`, `write`, `close`
- Command-line arguments
- Error handling with `errno`

### Exercises:
- 8-1 to 8-6 — pp. **202**

### Mini Projects:
- Build your own `cat`, `wc`, `cp`
- Mini-shell to execute commands

### Milestone:
- Can write low-level Unix-style CLI tools

---

## Week 8 – Final Review & Capstone

### Objectives:
- Review skipped exercises
- Refactor previous projects
- Combine everything into one large project

### Capstone Projects:
- Markdown or text file analyzer
- `nano`-like text editor (simplified)
- Basic file manager or task list

### Milestone:
- Confidence in using C for real-world programs
